# yazi config

Personal [Yazi](https://yazi-rs.github.io/) file manager configuration.

## Setup

Install plugins declared in `package.toml`:

```sh
ya pack -i
```

To update plugins later:

```sh
ya pack -u
```

### Tokyo Night theme

The `tokyo-night` flavor is not included in this repo. Clone it manually:

```sh
git clone https://github.com/BennyOe/tokyo-night.yazi.git ~/.config/yazi/flavors/tokyo-night.yazi
```
